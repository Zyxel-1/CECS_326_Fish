# Shared Memory and Semaphores
Fish Program demonstrating the usage of shared memory and Semaphores
## Getting Started
This program has been developed in Ubuntu 16.04.
### Pre-requisites
You must have GCC installed
### Running it
```
make
./swimlane
```
